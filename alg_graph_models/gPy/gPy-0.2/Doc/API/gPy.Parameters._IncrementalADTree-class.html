Initialise an _ADTree object

Parameters:
  • numvals - The ith element of this list is the number of values for the ith variable in the tree
  • records - The data as a list. Each element is a tuple of value indices, one for each variable, plus an extra count field as the final element.
  • depth - The depth of this tree within its containing CompactFactor. Also the index of the variable associated with the top node of this tree
  • rmin - If the number of records is below rmin then tree growing stops and records is stored. Note that a single record may represent many datapoints since all records have an extra 'count' field.
