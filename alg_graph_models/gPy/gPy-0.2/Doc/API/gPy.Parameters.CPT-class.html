bdeu_score(self, precision=1.0)

source code 

Return the BDeu component score corresponding to self

This is the score for the family represented by the parents and child of self and where the values of self are interpreted as data counts for the relevant instantiations.

The BDeu score for a Bayesian network is the sum of component scores for each of its CPTs.

llh(self, priors)

source code 

Return the marginal log-likelihood component score corresponding to self

This is the marginal log-likelihood component score for the family represented by the parents and child of self and where the values of self are interpreted as data counts for the relevant instantiations.

Since there are no constraints on priors likelihood equivalence may not obtain and so this score need not be a BDe score.

The marginal log-likelihood for a Bayesian network is the sum of component scores for each of its CPTs.

Parameters:
  • priors (CPT object) - A CPT object with the same structure as self where each value is the Dirichlet parameter for its corresponding instantiation.
Returns: Float
Marginal log-likelihood component score
Raises:
  • NameError - If gPyC was not successfully imported
