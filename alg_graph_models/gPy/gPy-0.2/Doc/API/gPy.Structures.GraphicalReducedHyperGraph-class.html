                        GraphicalReducedHyperGraph

For a hypergraph H to be graphical, the hyperedges of red(H) must be the cliques of an (undirected) graph.

A GraphicalReducedHyperGraph object is both reduced and graphical so its hyperedges are the cliques of an (undirected) graph.

Initialise a graphical reduced hypergraph

If hypergraph is indeed a hypergraph then the new GraphicalReducedHyperGraph object has identical attributes to hypergraph.

If hypergraph is not a hypergraph then it is assumed to be a collection of hyperedges, a HyperGraph object is created from these hyperedges as the first step of construction and execution proceeds exactly as if this HyperGraph object had been the original value of hypergraph. The default value for hypergraph constructs is an empty GraphicalReducedHyperGraph).
Parameters:
  • hypergraph (HyperGraph or iterable of iterables) - Hypergraph/hyperedges used to generate new reduced graphical hypergraph
  • check (Boolean) - Whether to check that hypergraph is reduced and graphical
Raises:
  • RedundancyError - If check=True and hypergraph is not reduced.
  • GraphicalityError - If check=True and hypergraph is not graphical.
