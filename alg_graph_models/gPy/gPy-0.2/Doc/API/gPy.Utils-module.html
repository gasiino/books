subsetn(set_in, n)

source code 

Generate subsets of size at most n

Actually set_in can be any iterable.

If n <= len(set_in)/2 tuples are returned. Each tuple is in ascending order. Otherwise a set is returned.

This is optimised for n < len(set)/2

Parameters:
  • set_in (Any iterable) - A 'set' of objects
  • n (Int) - Inclusive upper bound on size of subsets

source code  
Returns the average time elapsed in seconds.

Typical usage:
