Initialise a reduced hypergraph

If hypergraph is indeed a hypergraph then the new ReducedHyperGraph object has identical attributes to hypergraph.

If hypergraph is not a hypergraph then it is assumed to be a collection of hyperedges, a HyperGraph object is created from these hyperedges as the first step of construction and execution proceeds exactly as if this HyperGraph object had been the original value of hypergraph. The default value for hypergraph constructs is an empty ReducedHyperGraph).

Parameters:
  • hypergraph (HyperGraph or iterable of iterables) - Hypergraph/hyperedges used to generate new reduced hypergraph
  • check (Boolean) - In the case that modify=False, whether to check that hypergraph is reduced
  • modify (Boolean) - Whether hypergraph is to modified to be made reduced
  • trace (Boolean) - In the case where modify=True whether a mapping from removed redundant hyperedges to the set of hyperedges containing them should be created. If so, it is stored in a trace attribute of self
Raises:
  • RedundancyError - If check=True,modify=False and hypergraph is not reduced.
  • TypeError - If hypergraph is not a hypergraph or an iterable of hyperedges.
