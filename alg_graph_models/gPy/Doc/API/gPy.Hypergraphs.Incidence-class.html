Class Incidence

source code

Graphs and hypergraphs are implemented differently, so even though hypergraphs are a generalisation of graphs there is no direct inheritance relation defined for the relevant classes.

The name 'Incidence' should not be taken too seriously. There's nothing here specific to incidence structures.

This abstract class contains those few methods which are sufficiently abstract to work for both graphs and hypergraphs
