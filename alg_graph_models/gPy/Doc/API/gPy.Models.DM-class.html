Decomposable model

Interaction graph is decomposable (triangulated)

Note: There are no methods specific to this class. At present DM objects behave identically to HM objects so this class is only useful as a label claiming that a hierarchical model is decomposable. However there are methods for JFM objects which are just decomposable models with a particular join forest specified.
